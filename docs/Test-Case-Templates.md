# 🏥 AC-01 · 跌倒识别（FALL）

> 目标：在典型居家场景下模拟跌倒，系统需在 30 秒内触发 L3/L4 预警
> 测试覆盖：UT + API + IT + E2E

# 🔄 回归测试策略

1. **每次 PR** → 自动执行 UT 与基础 API 测试（CI 中自动运行）
2. **每日** → 夜间全量集成测试（IT），结果邮件通知测试团队
3. **每周** → 1 次完整 E2E 测试（含真实设备接入）
4. **每月** → 1 次性能基准测试 + 安全扫描
5. **每次 Release** → 全量回归测试（全部 AC-01 ~ AC-08 + SMOKE）

# 🛡 测试环境隔离规则

1. 测试环境数据库不与生产共享任何真实老人数据
2. 测试用老人数据为脱敏或虚构数据（姓名：测试-XXX，身份证哈希：000000-XXXX）
3. 测试环境 LLM Key 独立配置，有额度上限
4. 测试环境短信通道设为"沙箱模式"，不向真实号码发送
